ECMAScript 6 Hasher Plugin Fork on Github

es6-hasher is a plugin written in ECMAScript 6, which provides you a convenient way to store and manage key-value pairs in the URL hash.

es6-hasher can be translated to ECMAScript 5 scripts (by babel) working with most modern brwosers.

Functionalities & Examples

Suppose you already have an prefix hash 'oldhash'

window.location.hash = 'oldhash';

It's gonna remain its value during all actions.

(Configuration) Prefer replaceState or pushState ?

hasher.config({preferReplaceState: true});

or

hasher.config({preferReplaceState: false});

You can change this setting at any time.

Write a=valA to hash

hasher('a', 'valA');

Write b=valB to hash

hasher('b', 'valB');

Change key 'a' value to 'changedValA'

hasher('a', 'changedValA');

Write object {y:'valY', c:'valC'} to hash

hasher({y:'valY', c:'valC'});

Write empty key 'empty-key' to hash

hasher('empty-key', '');

Get the value of key 'a'

alert( hasher('a') );

Remove the key 'b'

hasher('b', null);

Sort hash by keys

hasher( hasher.symbol.sortByKeys );

or

hasher.sortByKeys();

Replace old hash with 'happyhash'

hasher( hasher.symbol.replacePrefixHash, 'happyhash' );

or

hasher.replacePrefixHash('happyhash');

Get current prefix hash

alert( hasher( hasher.symbol.prefixHash ) );

or

alert( hasher.prefixHash() );

Remove all empty keys in hash

hasher( hasher.symbol.removeEmptyKeys );

or

hasher.removeEmptyKeys();

Get all key-value pairs as an object

alert( hasher() )

Clean all keys

hasher( hasher.symbol.clean );

or

hasher.clean();

Clear everything in hash

hasher( hasher.symbol.clear );

or

hasher.clear();
